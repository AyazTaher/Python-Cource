# Python-Cource
دورة بايثون تتدرج من الأساسيات إلى المستويات المتقدمة، وتشمل تركيب اللغة، المتغيرات، النصوص، تراكيب البيانات مثل القوائم والقواميس، العمليات، الشروط، الحلقات، والدوال بأنواعها. كما تتناول قراءة الملفات وتنفيذ مشاريع تطبيقية في بيانات الطلاب، البيانات الصحية، وإدارة الأنظمة، لتأسيس متعلم قادر على بناء برامج واقعية.
